女媧實習心得

此文章同步刊登於 女媧資訊部落格

我是 Andy，從大三開始在女媧資訊擔任了兩年的實習生，很開心能夠遇到我的 mentor Bear 學長，讓我在這兩年的實習生活中過的充實又快樂。

兩年的實習時間相對於其他業界的實習計畫相比真的很長，涵蓋掉了一半的大學生活，但是短時間的實習，很難培養真正的「硬實力」，只要學會妥善的利用時間，在學校生活跟實習取得平衡點，會發現在這邊的實習是相當值得的。如果在過程中遇到問題，不論是課業上的問題，或是時間分配上的問題，Bear 也都會很願意提供協助。

在實習的前半年，培養的是實習生的硬實力，Bear 帶實習生從不藏私，並且這個實習生計畫也實行了很多年，累積下來了完整的教育訓練流程以及教材，教材每年都會更新，跟著 Bear 為實習生設計好的腳步一步一步走，可以學習到業界真正用到的 domain knowledge 以及扎實的 coding 技巧，比在學校所能接觸到的更多更廣，能夠幫助自己釐清未來的修課規劃，以及培養嚴謹程式思維與能力。

在實習的下一個階段，是整個實習計畫中收穫最大的階段，學長會開始讓我們實際參與各項專案，開始將所學轉化成產出，像是在實習的過程中，我接觸過珠寶公司的官網製作、與工研院合作的數據分析平台開發、公司電子報系統的升級維運，在這邊有機會接觸到各式各樣的專案。

正是因為這是 Bear 自行創業的小型公司，所以負責的工作常常會橫跨各個領域，包括從系統流程的設計到寫程式開發系統，會同時碰到前端以及後端，可能會需要了解從系統的上線流程到 Linux 到後續維護，有時需要寫寫資料庫 SQL 指令甚至到網站行銷 SEO，但是正是因為這些領域的包羅萬象，所以在這邊實習絕對不會無聊打雜，而是扎實而且業界正在使用的技術。

在這兩年的實習生活中，Bear 會很常關心實習生們的狀況，不論在工作或是學業方面，最後的半年因為 Covid-19 疫情的關係，我們改採取遠端的方式工作，但是仍然會得到老闆滿滿的關心以及偶爾打屁聊天，Bear 最近也升格成為一位爸爸，開始跟我們分享一些新手爸媽的經驗談，在這兩年中，就是跟著女媧的腳步一同學習成長，過程中是快樂的、扎實的也是值得的，會發現自己正朝著成為專業工程師的道路上邁進，很開心自己的大學生活有機會在女媧實習中度過。
